How to connect your bot to the channel:
1. Open Telegram → search @BotFather → send /newbot → copy the token into .env as TELEGRAM_BOT_TOKEN
2. Open your channel: https://t.me/+2VjqUrs0o_k2OGY1
3. Add the bot as Administrator of the channel
4. Get Chat ID: visit http://localhost:3000/api/admin/telegram-getid?token=YOUR_TOKEN
5. Paste Chat ID into .env as TELEGRAM_CHAT_ID → restart server
